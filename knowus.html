Pneumonia is an infection that inflames the air sacs in one or both lungs. The air sacs may fill with fluid or pus (purulent material), causing cough with phlegm or pus, fever, chills, and difficulty breathing. A variety of organisms, including bacteria, viruses and fungi, can cause pneumonia

The main objective for the project is to automate the process of detecting the disease (Pneumonia) by using Lung X-ray of the patient without any doctor intervention. This involves taking input (Lung X-ay) from patient and processing it by using thousands of records(X-rays) and by using Deep learning/transfer learning algorithms to arrive at a conclusion regarding the disease. This project involves end to end development of model and provides user interactive interface using web technologies.
